# Agenda de Contatos
Agenda de contatos em *flutter* utilizando a linguagem *dart*

## Funcionalidades
 - Utilização de Banco de Dados para armazenamento
 - Efetuar Ligações
 - Utilizar Camera e Galeria do Celular
